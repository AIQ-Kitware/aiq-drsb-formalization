\section{What is genuinely proved}
\begin{itemize}[leftmargin=2em]
\item The static value is no larger than the dynamic value by KL data processing under endpoint projection.
\item Static SB equals the entropic-OT value once the two reference endpoint measures are identified; this is a definitional rewrite rather than the analytic Gibbs-kernel theorem.
\item Algebraic conversions among control, log-potential, and value gradients are proved from explicit identification hypotheses.
\item Finite strictly positive Schrödinger potentials exist, are unique up to gauge, and gauge-normalized iterates converge.
\end{itemize}

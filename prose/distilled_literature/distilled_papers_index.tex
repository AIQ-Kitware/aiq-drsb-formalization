\section*{Purpose}
This index lists the standalone LaTeX notes in this directory.  Each note cites the paper it distills using \texttt{references.bib}.  The notes are designed to support a later pass where each Lean theorem is mapped to an exact paper theorem, lemma, equation, or proof paragraph.

\section*{Highest priority for the current Lean frontier}
The remaining finite Sinkhorn compactness work needs a proof that strict positivity of the kernel forces projective asymptotic regularity of alternating Sinkhorn updates.  The most relevant notes are:

\end{enumerate}
